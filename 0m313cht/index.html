二次规划（Quadratic programming）问题[1]

一个标准的二次规划问题的目标函数可以写成

min⁡uJ=12uHu+fTu \min_{\mathbf{u}}{J}=\frac{1}{2}\mathbf{u}\mathbf{H}\mathbf{u} + \mathbf{f}^T\mathbf{u} umin​J=21​uHu+fTu

其中

  • H\mathbf{H}H 是对称正定矩阵

提示

二次规划问题是一种特殊类型的最优化问题，当QQQ 为正定或正半定矩阵时，目标函数是凸的，此时问题有唯一的全局最优解(全局最小值)。

二次型目标函数前面加 12\frac{1}{2}21​ 主要是为了简化求导时的表达式，避免出现额外的系数，同时它也对问题的最优解没有实际影响。这种写法更多是数学推导和计算上的一种规范和习惯。

约束

QP问题的约束条件有三种

  • 不等式线性约束 MU<=bMU <= bMU<=b
  • Mequ=beqM_{eq}u = b_{eq}Meq​u=beq​
  • 上下限约束 LB<=u<=UBLB<=u<=UBLB<=u<=UB

最优控制问题[1:1]

最优控制是指在一定的约束条件下，通过优化系统的控制输入，使得系统在特定性能指标上实现最优表现的一类控制方法。

一个最优控制问题包含以下几个方面

  • 系统的数学模型(状态空间模型)
  • 目标值(参考值)
  • 性能指标(代价函数)
  • 约束条件

LQR VS MPC

特性LQRMPC
适用系统线性系统线性与非线性系统
目标函数二次型、时间无穷二次型、有限预测时间
优化问题的求解方式一次性求解，通过黎卡提方程得到反馈矩阵后，不再进行优化实时滚动优化，在每个时间步上重新求解优化问题
优化频率一次性（设计时求解一次，避让设定系统期望）每个时间步（实时求解，每次滚动窗口预测并优化）
计算复杂度低，控制过程中只需简单的矩阵乘法高，每个时间步需在线求解优化问题
约束处理无法直接处理，需要外部技术支持显式处理状态和控制输入约束

提示

MPC 与 LQR 的区别是增加了一个滚动优化控制（Receding Horizon Control）,滚动优化控制（Receding Horizon Control）是指从当前时刻到未来一时刻这一有限时间段内，根据当前状态以及对未来的预测
