点积（Dot product）的性质

a⃗⋅a⃗=∣a⃗∣∣b⃗∣cos⁡θ \vec{a} \cdot \vec{a} = |\vec{a}||\vec{b}|\cos{\theta} a⋅a=∣a∣∣b∣cosθ

当a⃗,b⃗\vec{a},\vec{b}a,b均为单位向量时，点积反应了向量间的夹角

  • 当 夹角接近90度时，趋近于0
  • 当 夹角>90度时，结果为负

向量叉积(cross product)的性质

向量外积 a×b\mathbf{a}\times \mathbf{b}a×b 或 a∧b\mathbf{a} \wedge \mathbf{b}a∧b

a×b=∥a∥∥b∥sin⁡(θ)n \mathbf{a}\times \mathbf{b}= \parallel \mathbf{a}\parallel \parallel \mathbf{b}\parallel \sin(\theta) \mathbf{n} a×b=∥a∥∥b∥sin(θ)n

考虑几何意义

  • 计算两个向量的旋转轴 n

四元数

注意

注意 Eigen::quaternion 有两种初始化方式，

  • Eigen::quaternion(Eigen::Vector4d(x,y,z,w))
  • Eigen::quaternion(w,x,y,z)

canonical form

A quaternion is in "canonical form" when its first element is nonnegative. 当四元数的第一个元素(w)为非负时，四元数处于“规范形式”。

q−1=q∗=(q0,−q1,−q2,−q3) \mathbf{q}^{−1}=\mathbf{q}^*=(q_0, -q_1, −q2, −q3_) q−1=q∗=(q0​,−q1​,−q2,−q3)​

Half-Way Vector Solution

Half-Way Quaternion Solution[1]

如何计算由向量 a -> b 的四元数 我们知道

a⃗⋅b⃗=∣a⃗∣∣b⃗∣cos⁡θa⃗×b⃗=∣a⃗∣∣b⃗∣sin⁡θn \begin{aligned} \vec{a} \cdot \vec{b} &= |\vec{a}| |\vec{b}| \cos \theta \\ \vec{a} \times \vec{b} &= |\vec{a}| |\vec{b}| \sin \theta \mathbf{n} \end{aligned} a⋅ba×b​=∣a∣∣b∣cosθ=∣a∣∣b∣sinθn​

不要单位向量忽略n

这里n为旋转轴 theta 为 旋转角,可表示成四元数q

q=[cos⁡θ2nsin⁡θ2] q = \begin{bmatrix} \cos{\frac{\theta}{2}}\\ \mathbf{n}\sin{\frac{\theta}{2}}\\ \end{bmatrix} q=[cos2θ​nsin2θ​​]

这里需要计算 cos⁡θ2\cos{\frac{\theta}{2}}cos2θ​ 和 sin⁡θ2\sin{\frac{\theta}{2}}sin2θ​，但是有取巧的办法。 即Half-Way Quaternion Solution，我们计算旋转两倍的四元数即

q=[cos⁡θnsin⁡θ] q = \begin{bmatrix} \cos{\theta}\\ \mathbf{n}\sin{\theta}\\ \end{bmatrix} q=[cosθnsinθ​]

接着找到他和0度之间一半的四元数（find the quaternion half-way between that and zero degrees.）。 零旋转的四元数为：

q2=[1000] q_2 = \begin{bmatrix} 1\\ 0\\ 0\\ 0 \end{bmatrix} q2​=​1000​​

为什么可以使用加法计算一半旋转角的四元数？

接着我们可以仅通过加法计算,这里需要q 和 q_2 有相同的尺度，q的大小为 ∣a⃗∣∣b⃗∣|\vec{a}||\vec{b}|∣a∣∣b∣ ,将q_2写成

q2=[∣a⃗∣∣b⃗∣000]=[∣a⃗∣2∣b⃗∣2000] q_2 = \begin{bmatrix} |\vec{a}| |\vec{b}|\\ 0\\ 0\\ 0 \end{bmatrix} =\begin{bmatrix} \sqrt{|\vec{a}|^2 |\vec{b}|^2}\\ 0\\ 0\\ 0 \end{bmatrix} q2​=​∣a∣∣b∣000​​=​∣a∣2∣b∣2​000​​

最后的形式比上一步少算了一个根号. 代码对照

欧拉方程的推导

坐标系

首先明确两个参考系：

  • 空间系（space frame）: 一个被固定在空间中的惯性系 S0S_0S0​ 。
  • 物理系（body frame）：一个被固定在转动的刚体上的，自身会转动的非惯性系 SSS 。

推导

设物体的角速度为 ω=(ω1,ω2,ω3)\mathbf{\omega}=(\omega_1,\omega_2,\omega_3)ω=(ω1​,ω2​,ω3​) ,转动惯量分别为 λ1,λ2,λ3\lambda_1,\lambda_2,\lambda_3λ1​,λ2​,λ3​, 则在 body frame 里，物体的角动量为

L=(λ1ω1,λ2ω2,λ3ω3) \mathbf{L}=(\lambda_1 \omega_1,\lambda_2 \omega_2,\lambda_3 \omega_3) L=(λ1​ω1​,λ2​ω2​,λ3​ω3​)

力矩为 Γ\mathbf{\Gamma}Γ ,则在space frame里，有

Γ=(dLdt)space \mathbf{\Gamma} = (\frac{d\mathbf{L}}{dt})_{space} Γ=(dtdL​)space​

提示

一个重要的结论：向量Q\mathbf{Q}Q 在惯性系S0S_0S0​ 与在非惯性系SSS中的时间导数的关系是：

(dQdt)S0=(dQdt)S+Ω×Q \left(\frac{d\mathbf{Q}}{dt}\right)_{S_0}= \left(\frac{d\mathbf{Q}}{dt}\right)_{S}+\mathbf{\Omega}\times \mathbf{Q} (dtdQ​)S0​​=(dtdQ​)S​+Ω×Q

其中Ω\mathbf{\Omega}Ω 是body frame下的角速度

将L\mathbf{L}L 带入Q\mathbf{Q}Q，得到

(dLdt)S0=(dLdt)S+ω×L=L˙+ω×L \left(\frac{d\mathbf{L}}{dt}\right)_{S_0}= \left(\frac{d\mathbf{L}}{dt}\right)_{S}+\mathbf{\omega}\times \mathbf{L} =\mathbf{\dot{L}}+\mathbf{\omega}\times \mathbf{L} (dtdL​)S0​​=(dtdL​)S​+ω×L=L˙+ω×L

函数有唯一极值点的条件

对于一元函数

对于一个一元函数f(x)f(x)f(x)，唯一极值点的条件主要涉及一阶导数和二阶导数。

局部极值点的判定

条件一：必要条件 在x0x_0x0​处的一阶导为0即(f′′(x)=0f^{''}(x)=0f′′(x)=0)或一阶导不存在。

条件二：二阶导数判定

  • 如果函数的二阶导数 f′′(x)f^{''}(x)f′′(x) 在某点x0x_0x0​处是正的,即f′′(x)>0f^{''}(x)>0f′′(x)>0,则x0x_0x0​是一个局部极小值点。
  • 如果函数的二阶导数 f′′(x)<0f^{''}(x)<0f′′(x)<0,则x0x_0x0​是一个局部极大值点。
  • 如果 f′′(x)=0f^{''}(x)=0f′′(x)=0，则需要进一步分析更高阶导数。

唯一极值点的条件

函数的单峰性：函数具有唯一的极大值或极小值，通常表现为单调递增到一个点后变为单调递减（或相反），这种性质通常与函数的凸性或凹性有关。

  • 如果函数是严格凸的，即 f′′(x)>0f^{''}(x)>0f′′(x)>0 对于所有 x∈Rx\in\mathbb{R}x∈R,则函数只有一个极小值点。
  • 如果函数是严格凹的，即 f′′(x)<0f^{''}(x)<0f′′(x)<0 对于所有 x∈Rx\in\mathbb{R}x∈R,则函数只有一个极大值点。

对于多元函数

局部极值点的判定

条件一：必要条件 梯度向量为0。 条件二: Hessian 矩阵判断法,通过Hessian矩阵的正定性或者负定性判定:

  • 如果 Hessian 矩阵在某点正定，则该点是局部极小值点。
  • 如果 Hessian 矩阵在某点负定，则该点是局部极大值点。
  • 如果 Hessian 矩阵既非正定也非负定，可能存在鞍点。

唯一极值点的条件

  • 严格凸性：如果函数 f(x1,x2,...,xn)f(x_1,x_2,...,x_n)f(x1​,x2​,...,xn​) 是严格凸的（即 Hessian 矩阵在定义域内正定），则该函数只有一个全局极小值点。
  • 严格凹性：如果函数是严格凹的（即 Hessian 矩阵在定义域内负定），则该函数只有一个全局极大值点。

提示

在优化问题中，通常需要求极小值，例如在QP问题中，需要保证 Hessian 矩阵是一个正定矩阵。（正定矩阵首先是对称的）
