## 安装

filebrowser 的安装非常简单

## 快速使用

```bash
filebrowser -r /file-storage/ -p 2803 -a xxx.xxx.xxx.189
# 加载已有配置
filebrowser -d filebrowser.db
```

## 使用配置

```bash
filebrowser -d filebrowser.db config set -a 172.16.227.189
```
